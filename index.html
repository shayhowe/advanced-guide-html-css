Homework

  1. Read this weeks, as well as next weeks, lesson on An Advanced Guide to HTML & CSS
  2. Create a personal website, including Home, About, and Experiments pages
    • Include a header, global navigation, footer, and at least two columns
  3. Add the appropriate content on the Home and About pages
    • On the Home page introduce visitors to your website and loop in all of your other pages
    • On the About page briefly describe yourself, perhaps including your resume
  4. On the Experiments page create a set of alert messages, using modular styles based on what was learned from building modular buttons
    • Alert messages should include styles for information, success, warning, and error messages
    • Inspiration for different alert message styles may be found on Dribbble
  5. Create an account on GitHub and try to install Git on your computer
